# Xcode Cloud

> Xcode Cloud 是一种由 Apple 提供的服务，可以帮助开发人员将 Xcode 项目托管到云端并在云端进行构建和测试。它可以帮助团队更快地提交和测试代码，并减少本地构建机器的需求。它还可以帮助团队更好地组织和管理构建和测试工作流程。
>

## TestFlight 如何区分 beta 环境 和 prod 环境
在使用 TestFlight 测试应用程序时，您可以通过设置不同的内部或外部测试组来区分 beta 环境和生产环境。



对于内部测试，您可以在 App Store Connect 中创建不同的测试组，并将其分配给特定的测试人员。例如，您可以创建一个“beta 测试组”，并将其分配给团队中的测试人员。



对于外部测试，您可以在 App Store Connect 中创建不同的测试组，并将其分配给特定的外部测试人员。例如，您可以创建一个“生产测试组”，并将其分配给顾客或其他外部测试人员。



在这种情况下，您可以更好的控制和管理测试人员的权限，最后在生产环境中发布的版本是经过验证的。
