# rax - 快速构建多端应用

[https://rax.js.org/](https://rax.js.org/)



## <font style="color:rgb(13, 26, 38);">主要特性</font>
+ <font style="color:rgb(51, 51, 51);">Rax 语法层面以 React 为标准，可以使用 Hooks、Context 等 80% 以上支持度的 React API</font>
+ <font style="color:rgb(51, 51, 51);">官方配套的研发框架 Rax App，支持 TypeScript、Less/Sass 等基础工程能力，同时支持 MPA、SPA、SSR 多种能力</font>
+ <font style="color:rgb(51, 51, 51);">支持通过完整的 Rax 语法开发跨支付宝/微信/字节等不同厂商的小程序，同时可降级到 Web</font>
+ <font style="color:rgb(51, 51, 51);">基于 Web 标准支持跨多容器的跨端应用，包含 Web 应用、Flutter 应用（Kraken）、Weex 应用</font>
+ <font style="color:rgb(51, 51, 51);">丰富的跨端生态，比如跨端组件 Fusion Mobile，跨端 API Uni API</font>
