# Android四大组件

<font style="color:rgb(34, 34, 34);">从事Android开发，我们都知道Android有四大组件， 分别是：</font>

+ <font style="color:rgb(34, 34, 34);">活动（Activity），用于表现功能，是用户操作的可视化界面，它为用户提供了一个完成操作指令的窗口；</font>
+ <font style="color:rgb(34, 34, 34);">服务（Service），后台运行服务，不提供界面呈现；</font>
+ <font style="color:rgb(34, 34, 34);">广播接受者（Broadcast Receive），用于接收广播；</font>
+ <font style="color:rgb(34, 34, 34);">内容提供者（Content Provider），支持多个应用中存储和读取数据，相当于数据库。</font>
