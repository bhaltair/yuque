# 钉钉群添加资讯订阅机器人

## 步骤
1. 钉钉群新建机器人
2. 得到web hook地址
3. postman调试一下 Link Markdown的推送
4. 去rss hub 找到想订阅的资讯
5. ifttt配置





## 钉钉机器人发送一条带交互的按钮


```plain
{
  "chat_id": 131093671,         # 对话id。若是频道则为“@channel_id”
  "text": "**hi**",             # 发送的信息正文。可以使用有限的几种 Markdown/HTML 进行标记
  "reply_markup": {             # 按钮
    "inline_keyboard": [[{      # 附在信息之后的按钮。是个二维数组是因为可以有若干行，每行有若干个按钮
      "text":"go",              # 按钮文字内容
      "url":"www.google.com"    # 点击按钮跳转的目标链接
  }]]
  },
  "parse_mode":"Markdown"       # 消息的解析模式
}
```



## rss资讯


[https://docs.rsshub.app/](https://docs.rsshub.app/)



## ifttt


条件 选择rss feed

then 选择web hook 地址选择钉钉机器人的地址 选择POST请求 报文 配置Link或者Markdown模式
