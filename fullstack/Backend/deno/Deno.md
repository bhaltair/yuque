# Deno 1.9
[Deno 1.9 发布](https://deno.com/blog/v1.9)，此版本包含诸多新功能、性能改进以及错误修复：

+ [全新的原生 HTTP/2 Web 服务器](https://deno.com/blog/v1.9#native-http%2F2-web-server)。
+ [使用 serde_v8 提升 Rust 性能](https://deno.com/blog/v1.9#faster-calls-into-rust-with-serde_v8)。
+ [Blob URL 支持和 fetch 改进](https://deno.com/blog/v1.9#blob-url-support-%26-improvements-to-fetch)。
+ [LSP 中的目录自动补全功能](https://deno.com/blog/v1.9#import-completions-in-the-lsp)。
+ [交互式权限提示](https://deno.com/blog/v1.9#allow-lists-for---allow-env-and---allow-run)。
