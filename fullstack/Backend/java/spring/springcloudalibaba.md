# spring cloud alibaba

# 已经包含的组件


**<font style="color:rgb(36, 41, 46);">Sentinel</font>**

<font style="color:rgb(36, 41, 46);">阿里巴巴开源产品，把流量作为切入点，从流量控制、熔断降级、系统负载保护等多个维度保护服务的稳定性。</font>

**<font style="color:rgb(36, 41, 46);">Nacos</font>**

<font style="color:rgb(36, 41, 46);">阿里巴巴开源产品，一个更易于构建云原生应用的动态服务发现、配置管理和服务管理平台。</font>

**<font style="color:rgb(36, 41, 46);">RocketMQ</font>**

<font style="color:rgb(36, 41, 46);">Apache RocketMQ</font><font style="color:rgb(36, 41, 46);">™</font><font style="color:rgb(36, 41, 46);"> 基于 Java 的高性能、高吞吐量的分布式消息和流计算平台。</font>

**<font style="color:rgb(36, 41, 46);">Dubbo</font>**

<font style="color:rgb(36, 41, 46);">Apache Dubbo</font><font style="color:rgb(36, 41, 46);">™</font><font style="color:rgb(36, 41, 46);"> 是一款高性能 Java RPC 框架。</font>

**<font style="color:rgb(36, 41, 46);">Seata</font>**

<font style="color:rgb(36, 41, 46);">阿里巴巴开源产品，一个易于使用的高性能微服务分布式事务解决方案。</font>

**<font style="color:rgb(36, 41, 46);">Alibaba Cloud OSS</font>**

<font style="color:rgb(36, 41, 46);">阿里云对象存储服务（Object Storage Service，简称 OSS），是阿里云提供的海量、安全、低成本、高可靠的云存储服务。您可以在任何应用、任何时间、任何地点存储和访问任意类型的数据。</font>

**<font style="color:rgb(36, 41, 46);">Alibaba Cloud SchedulerX</font>**

<font style="color:rgb(36, 41, 46);">阿里中间件团队开发的一款分布式任务调度产品，支持周期性的任务与固定时间点触发任务。</font>

**<font style="color:rgb(36, 41, 46);">Alibaba Cloud SMS</font>**

<font style="color:rgb(36, 41, 46);">覆盖全球的短信服务，友好、高效、智能的互联化通讯能力，帮助企业迅速搭建客户触达通道。</font>
