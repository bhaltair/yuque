# 变量的解构赋值

## 
<font style="color:#DF2A3F;">如果给变量设置了默认值，则只有在对应属性值为undefined时才会使用默认值。如果对应属性值为null，则会将其作为有效值进行赋值</font>



## 嵌套解构


```javascript
const user = {

 id: 123,

 name: 'hehe'

};

const {

   education: {

       degree

   } = {}

} = user;

console.log(degree); //prints: undefined
```
