# lodash api 学习

## lodash api学习


_.chunk  
_.chunk(array, [size=1])  
将数组进行分块，按照size指定的长度，默认长度1



_.compact  
_.compact(array)  
剔除数组中没有意义的值，比如false, null, 0, "", undefined 和 NaN



_.concat  
_.concat(array, [values])  
创建一个新的数组来保存原始数组，增加值/数组之后的结果



_.flatten  
_.flatten(array)  
这个函数的作用是将array减少一个维度。
