# 变量类型

## 类型转化


### 转boolean


undefined， null， false， NaN， ''， 0， -0都会转化为false，其他都是true



## 判断类型


### typeof


对于 null 来说，虽然它是基本类型，但是会显示 object，这是一个存在很久了的 Bug

其他的类型都能很好的判断





### instanceof
> 判断一个变量是否是数组，使用typeof无法判断
>
> 可以使用[1, 2] instanceof Array来判断
>
